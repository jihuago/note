* taskwarrior
这是喜欢用命令行的同学福音，taskwarrior是可以在命令行添加、删除任务。

    官网：https://taskwarrior.org/
    
    * 常用命令：
    
    新增一个任务“写作”
    ```
    $ task add 写作  
    ```
    查看任务
    ```
    $ task list
    ```
    完成任务2
    ```
    $ task 2 done
    ```
    
    创建一个优先级高的任务
    ```
    $ task add priority:H taskwarrior命令
    ```
